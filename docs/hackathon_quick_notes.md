# AI Health Guardian Hackathon Quick Notes

## Project identity
AI Health Guardian is a responsive healthtech app that combines:
- symptom checker
- patient data visualization
- disease trend dashboard
- skin condition classifier

## One-line pitch
We built a preventive health platform that helps users understand symptoms, track their vitals visually, monitor disease trends, and get AI-assisted skin awareness support.

## Problem
- health information is hard to interpret
- users miss trends in vitals over time
- disease patterns are not easy to understand visually
- image-based health concerns often need simple first-level guidance

## Solution
- structured symptom input
- local health dashboards and trend charts
- disease trend analytics with filters
- Gemini-powered summaries and image analysis

## Where Gemini is used
- symptom analysis
- skin image analysis
- medicine explanation
- disease trend summary

## 4 module explanation
### 1. Symptom Checker
- user selects symptoms
- app analyzes combinations
- app returns likely guidance with caution

### 2. Patient Data Visualization
- app plots BMI, sleep, steps, and heart-risk trends
- user can notice abnormal patterns early

### 3. Disease Trend Dashboard
- local disease dataset
- filter by disease, region, and time
- show spikes, peaks, and region comparison
- Gemini turns trend numbers into summary points

### 4. Skin Condition Classifier
- user uploads image
- Gemini analyzes visible skin condition clues
- app provides awareness-oriented result

## Innovation points
- one app covers 4 healthtech blueprints
- responsive web UI
- multimodal AI use
- health dashboard + analytics + image model in one workflow

## Demo order
1. dashboard overview
2. symptom checker
3. patient trend charts
4. disease trend dashboard
5. skin condition classifier
6. bonus: reports / emergency / chatbot

## Short judge answers
### Why this project?
Because health data is more useful when it is interpreted, visualized, and summarized clearly.

### Why AI?
AI helps convert raw symptoms, charts, and images into understandable guidance.

### Why is this useful?
It supports early awareness, self-monitoring, and easier access to health information.

### What makes it hackathon-worthy?
It combines data visualization, rule-guided health input, image analysis, and AI summarization in one usable product.

## 60-second close
AI Health Guardian shows how healthtech can become more accessible by combining structured health inputs, visual trend analysis, and Gemini-powered intelligence in one responsive app. It is a practical prototype for early awareness, monitoring, and digital preventive care.
